Паттерн выхода из сессии

Единый компонент: кнопка logout в хедере + confirm modal

Кнопка "Выйти" есть на каждом защищенном экране.
После клика открывается окно подтверждения.
При подтверждении сессия завершается и токены очищаются.
Access token обновляется автоматически без отдельного UI.
